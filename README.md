# Название
*Простой AR-шутер под Android.*

## Features:

Стрельба из фиксированного на камере пистолета.

Спавн монстров в ограниченной области вокруг камеры

Счетчик убийств

Звуки выстрелов и смерти

## Instruments:

AR

Audio Source

TextMesh Pro

VFX-анимация (взрыв врага при убийстве)

## Toolkit:

C#, Unity
