# Название
*Простой AR-шутер под Android.*

## Features:

Стрельба из фиксированного к камере пистолета.

Спавн монстров в ограниченной области вокруг камеры

Счетчик убийств

Звуки выстрелов и убийства

## Instruments:

AR

Audio Source

TextMesh Pro

VFX (взрыв врага при убийстве)

## Toolkit:

C#, Unity
